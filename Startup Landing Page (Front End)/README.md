# Startup Brand — Landing Page (Front End)

Simple responsive landing page built with Bootstrap. Includes a separate Team Members page.

Files:
- `index.html` — Main landing page with sections: Header, Features, Testimonial, Price List, CTA, Social Link.
- `team.html` — About and Team Members page (responsive grid: 3 columns desktop, 2 tablet, 1 mobile).
- `css/style.css` — Small custom styles.

How to use:
1. Open `index.html` in a browser (double-click or use Live Server).
2. Use the navbar to jump to sections or click "Team Members" to open the separate page.

Notes:
- Built using Bootstrap 5 CDN and Bootstrap Icons CDN. No build tool required.
- Replace placeholder images / links with real assets and GitHub/Resume links.
